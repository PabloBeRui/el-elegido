Instalado react-router-dom

Instalado  MUI (conocido anteriormente por Material-ui), una biblioteca de componentes de interfaz de usuario diseñada para react

npm install @mui/material @emotion/react @emotion/styled

Instalo la biblioteca de iconos de mui

npm install @mui/icons-material


Instalo la biblioteca react-slick y sus dependencias de estilo slick-carousel, para crear un carrusel de imágenes
npm install react-slick slick-carousel
